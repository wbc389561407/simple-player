# 简单的播放器

#### 介绍
一个简单的播放器，我起了个名字叫做《sp播放器》，是simple-player的意思

# 使用说明
### 一、下载
https://github.com/wbc389561407/simple-player/releases/tag/spv1
![img_4.png](img_4.png)
### 二、解压
下载所有文件，解压到一个文件夹中,进行解压
### 三、运行
解压后，运行simple-player.exe
### 四、使用
可以播放普通文件,如果需要播放加密文件，需要激活
